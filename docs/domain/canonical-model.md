# Canonical Domain Model (Draft)

## Core Concepts

### Canonical Object
Base structure for all major entities:
- id
- type
- status
- effectiveDates
- auditFields
- metadata
- tags/dimensions

### Event
Represents a financial-flavored transaction.

### Agent
Represents a role-bearing entity:
- Person
- Organization
- Supplier
- Student
- Employee

### Role
Defines contextual permissions and responsibilities.
