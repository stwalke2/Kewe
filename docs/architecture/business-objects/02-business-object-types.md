# Business Object Types

## Responsibility Objects
- Cost Center
- Department
- Academic Unit
- Organization
- Company
- Location

## Funding Source (Superclass)
Subtypes:
- Fund
- Grant
- Endowment
- Expendable Gift
- Appropriation
- Debt
- Cost Share

## Activity Objects
- Project
- Program
- Activity
- Event
- Function (primarily reporting-driven)

## Agent Objects (REA Model)
Superclass: Agent
Subtypes:
- Person
- Organization
- Other Agent

## Financial Structure Objects
- Ledger Account (Natural Account)
- Spend Category
- Revenue Category
